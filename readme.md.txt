# Clinical Periodontal Chart Web App

This is a React-based web application to generate **clinical periodontal charts** with:

- Upper and lower arches in **classical clinical layout**  
- **Editable probing depths** for each tooth (6 sites per tooth: MB, B, DB, ML, L, DL)  
- Support for **missing teeth**  
- **User-defined probing depth weights** (1â€“4 mm)  
- **Print-friendly layout**
